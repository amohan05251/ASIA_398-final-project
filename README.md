# ASIA_398-final-project
## Project type: Ramayana Interactive Website 

This project was created as my final assignment for an Asian Studies course at the University of British Columbia. Students were encouraged to present their work in any creative format, and as a Computer Science student, I chose to design and develop an interactive website using HTML, CSS, and JavaScript.

The website explores the Ramayana, one of India's most influential epics, by presenting its major themes, characters, events, and cultural significance in an engaging and visually interactive format. Rather than submitting a traditional essay or presentation, I used web development to create an educational experience that allows users to navigate different aspects of the epic through a modern interface.

This project combines my interests in technology and the humanities, demonstrating how web development can be used as a medium for storytelling and education. It also reflects my ability to apply technical skills to communicate complex cultural and historical concepts in an accessible and creative way.

Live Website: https://amohan05251.github.io/ASIA_398-final-project/
### Technologies Used

- HTML5
- CSS
 
### Key Features

Interactive navigation between different sections of the Ramayana
Visually engaging layouts and animations
Responsive web design
Educational content presented through a modern website interface

This project highlights the intersection of computer science and the humanities, showcasing how software development can be used to create meaningful educational experiences.
